Oder-Verknüpfung

Werden mehrere Suchbegriffe eingegeben, erfolgt intern eine Oder-Verknüpfung. D.h. es werden alle Seiten ausgegeben, auf denen wenigstens einer der Suchbegriffe vorkommt. Eine Und-Verknüpfung ist z.Zt. nicht möglich.

Ausschluss von der Suche

Um einen Suchbegriff auszuschliessen, setzen Sie davor ein "-" (Minus), dann werden Seiten unterdrückt, die diesen Suchbegriff enthalten.

Bereiche von Indizierung ausschliessen

Textbereiche können von der Indizierung ausgeschlossen werden, indem sie in
<phpcms:noindex>Ausschlusstags</phpcms:noindex>
gesetzt werden.

Versuchen Sie selbst Ihr Glück, und suchen Sie nach „Ausschlusstag”!

Die Indexdateien für diese Suche befinden sich im Verzeichnis $home/suche.
